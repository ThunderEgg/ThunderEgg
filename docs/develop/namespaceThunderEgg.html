using ThunderEgg::IsNeumannFunc = typedef std::function<bool(Side<D> s, const std::array<double, D> &lower, const std::array<double, D> &upper)>

This function is used to set the boundary condition types of the solver.

This function will only be called on the boundaries of the domain.
