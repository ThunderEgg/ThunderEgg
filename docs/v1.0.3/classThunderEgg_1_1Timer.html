Detailed Description

Class for keeping track of parallel timings.

This class will keep track of timings and present them in a nice format.

This class also keeps track of nested timings. For following 

    timer.start("A");
    timer.start("B");
    timer.stop("B");
    timer.stop("A");

outputs 

    A
    -
    time (sec): 1.0


    A -> B
    ------
    time(sec):
